# Personal Web — 个人网站

## 核心目标

展示个人 demo 项目和文章内容的个人网站。

## 术语

| 术语 | 定义 |
|------|------|
| **Demo 项目** | 可在页面内直接交互的前端演示（React 组件、Canvas 动画等） |
| **独立项目** | 需要外部访问的项目（GitHub 项目、API 服务等），以介绍卡片 + 链接形式展示 |
| **项目** | Demo 项目 + 独立项目的统称 |

## 决策记录

| 决策 | 选择 | 理由 |
|------|------|------|
| 技术栈 | Next.js + TypeScript (App Router) | MDX 写文章 + React 组件做 demo，两头都好 |
| 部署 | Vercel | Next.js 原生支持，零配置 |
| 内容管理 | 纯本地 MDX 文件 | 零数据库，Git 版本控制 |
| UI 框架 | Tailwind CSS + shadcn/ui | 快速出漂亮页面 |
| 视觉风格 | 复刻 huasheng.ai | 深色暖调 + 金色点缀 + 电影感排版 |
| 语言 | 纯中文 | 目标受众为国内开发者 |
| 分类/标签 | 暂不做 | 初期内容量少，不需要 |
| 暗色模式 | 暂不做 | 网站本身就是深色主题 |
| 内容初始 | Placeholder | 先跑通框架再填内容 |

## 视觉语言（huasheng.ai 复刻）

- **主背景**: `#0E0D0B`（深暖黑）
- **主文字**: `#F2EADC`（奶油色）
- **点缀色**: `#BC9B6A`（金色）
- **浅色区块背景**: `#F2EADC` / `#ECE4D4`
- **深色区块文字**: `#1B1611`
- **辅助文字**: `rgba(242,234,220,0.55)`
- **字体**: Display 字体（标题 italic）、等宽字体（标签全大写 + 宽字距）、Sans-serif（正文）
- **排版**: 区块交替深色/浅色，1px 分割线，大面积留白，电影叙事标签（"REEL 01 · SCENE 001"）
- **交互**: hover 颜色渐变、入场动画

## 网站结构

```
/                    → 首页（自我介绍 + 精选项目 + 最近文章）
/projects            → 项目列表页
/projects/[slug]     → 单个项目详情 / 交互式 demo
/blog                → 文章列表页
/blog/[slug]         → 文章详情页（MDX）
```
